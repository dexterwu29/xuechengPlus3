## 一、角色与权限说明

| 角色 | 说明 | 可见菜单 |
|------|------|----------|
| **visitor** | 访客/学员 | 首页、课程（免费/精选/新上架）、分享论坛、关于我们 |
| **teacher** | 机构教师 | 上述 + 管理页面（课程管理、媒资管理、备案管理） |
| **admin** | 机构管理员 | 同 teacher |
| **super_admin** | 超级管理员 | 同 teacher，可跨机构操作 |

**测试账号**：参见 `TEST-ACCOUNTS-local-example.md`，密码均为 `password`。

## 四、专项功能验证

### 4.1 多封面（文档 21 第二 2.1/2.3）

| 验证项 | 操作 | 预期 |
|--------|------|------|
| 上传数量 | 课程编辑-封面上传 | 最多 3 张 |
| 默认选择 | 选择 1 张为默认 | 列表/详情主图为该张 |
| 详情展示 | 课程详情页 | 多图展示区域 |

### 4.2 营销校验（文档 21 第三 3.1）

| 验证项 | 操作 | 预期 |
|--------|------|------|
| QQ 必填 | 提交时 QQ 为空 | 校验失败 |
| 微信必填 | 提交时微信为空 | 校验失败 |
| 手机必填 | 提交时手机为空 | 校验失败 |
| 手机格式 | 输入非 1[3-9] 开头 | 校验失败 |

### 4.3 媒资路径与秒传（文档 21 第五）

| 验证项 | 操作 | 预期 |
|--------|------|------|
| 路径格式 | 上传后查 MinIO | `{md5[0]}/{md5[1]}/{md5}/{md5}_{fileName}.{ext}` |
| 小文件秒传 | 上传相同 MD5 文件 | 直接返回 fileId，无实际上传 |
| 大文件秒传 | 分片 init 时 MD5 已存在 | 返回 instant+fileId，跳过上传 |

### 4.4 审核流程（文档 21 第七）

| 验证项 | 操作 | 预期 |
|--------|------|------|
| 教师提交 | 教师点击提交 | 草稿→待审核 |
| 管理员提交 | 管理员点击提交 | 直接已发布 |
| 已发布修订 | 教师修改已发布课程再提交 | 进入待审核 |

## 五、测试执行建议

1. **环境**：本地 `spring-boot:run -Dspring-boot.run.profiles=local`，前端 `npm run dev`
2. **账号**：使用 `TEST-ACCOUNTS-local-example.md` 中账号，切换不同角色
3. **机构**：`localStorage.setItem('companyId','1')` 或 `2` 切换机构
4. **记录**：建议用表格记录通过/失败，便于回归
