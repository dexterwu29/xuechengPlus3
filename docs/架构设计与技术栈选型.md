# 学成在线三代 架构设计与技术栈选型

## 一、技术栈版本选型

### 1.1 后端核心

| 组件 | 版本 | 说明 |
|------|------|------|
| Java | 17 (LTS) | 与 Spring Boot 3.x 匹配 |
| Spring Boot | 3.4.x | 当前稳定，支持 Spring AI 1.0.0-m4 |
| Spring Cloud | 2024.0.1 (Moorgate) | 基于 Spring Boot 3.4.3，与 3.4.x 兼容 |
| Spring Framework | 6.2.x | 随 Spring Boot 3.4 引入 |

### 1.2 AI 集成方案（二选一或并存）

| 方案 | 适用场景 | 依赖 |
|------|----------|------|
| **Spring AI** | 与 Spring 生态深度集成，ChatClient、FunctionCallback、RAG | `spring-ai-*` 系列 |
| **LangChain4j** | 声明式 AI Service、多模型、工具链 | `langchain4j-*-spring-boot-starter` |

**Spring AI** 已随 Spring Boot 3.4 提供，适合快速接入；**LangChain4j** 适合需要更灵活 RAG、多模型切换的场景。

### 1.3 前端

| 组件 | 版本 |
|------|------|
| Node.js | ≥ 20 (推荐 20.19.5 LTS) |
| Vue | 3.x |
| 构建工具 | Vite 5.x |

## 三、父 POM 与 BOM 依赖

### 3.1 父 POM (`backend/pom.xml`)

```xml
<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0
         https://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>3.4.3</version>
        <relativePath/>
    </parent>

    <groupId>com.xuecheng</groupId>
    <artifactId>xuecheng-plus3</artifactId>
    <version>1.0.0-SNAPSHOT</version>
    <packaging>pom</packaging>
    <name>xuecheng-plus3</name>
    <description>学成在线三代 微服务父工程</description>

    <properties>
        <java.version>17</java.version>
        <spring-cloud.version>2024.0.1</spring-cloud.version>
        <spring-ai.version>1.0.0-M4</spring-ai.version>
        <langchain4j.version>1.0.0-beta1</langchain4j.version>
    </properties>

    <dependencyManagement>
        <dependencies>
            <!-- Spring Cloud BOM -->
            <dependency>
                <groupId>org.springframework.cloud</groupId>
                <artifactId>spring-cloud-dependencies</artifactId>
                <version>${spring-cloud.version}</version>
                <type>pom</type>
                <scope>import</scope>
            </dependency>
            <!-- Spring AI BOM（可选） -->
            <dependency>
                <groupId>org.springframework.ai</groupId>
                <artifactId>spring-ai-bom</artifactId>
                <version>${spring-ai.version}</version>
                <type>pom</type>
                <scope>import</scope>
            </dependency>
        </dependencies>
    </dependencyManagement>

    <modules>
        <module>xc-plus-gateway</module>
        <module>xc-plus-content-service</module>
        <module>xc-plus-media-service</module>
        <module>xc-plus-system-service</module>
        <module>xc-plus-auth-service</module>
        <module>xc-plus-learning-service</module>
        <!-- 其他微服务按需添加 -->
    </modules>
</project>
```

### 3.2 微服务通用依赖（子模块 `pom.xml` 示例）

```xml
<dependencies>
```

### 3.3 Spring Cloud Alibaba 版本

Spring Cloud 2024.0.x 需搭配 **Spring Cloud Alibaba 2023.0.x** 使用 Nacos：

```xml
<dependency>
    <groupId>com.alibaba.cloud</groupId>
    <artifactId>spring-cloud-alibaba-dependencies</artifactId>
    <version>2023.0.1.2</version>
    <type>pom</type>
    <scope>import</scope>
</dependency>
```

## 四、二代与三代组件对照

| 二代（黑马 192.168.101.65） | 三代（腾讯云） | 说明 |
|-----------------------------|----------------|------|
| nacos-server:1.4.1 | nacos:2.4+ | 支持 Spring Boot 3 客户端 |
| mysql:8.0.26 | mysql:8.0 | 保持 8.x |
| redis:6.2.7 | redis:7.x | 建议升级 |
| minio:RELEASE.2022-09-07 | minio:latest | 建议使用较新版本 |
| elasticsearch:7.12.1 | elasticsearch:8.x 或 7.17 | 按需选择 |
| rabbitmq:3.8.34 | rabbitmq:3.13+ | 建议升级 |
| xxl-job-admin:2.3.1 | xxl-job:2.4+ | 兼容 Spring Boot 3 |
